Multiple Gifts for Spouses - v1.0.0
10/07/2018

Author:
	Redwood

Features:
	Spouses can be gifted multiple times per day (twice by default)
	Editable 'config.json' file for changing the gift limit

Install:
	Install latest version of SMAPI (2.7 as of writing)
	Download and unzip mod folder directly into Stardew Valley/Mods folder
	Run Stardew Valley using SMAPI

Un-install:
	Remove mod folder from Stardew Valley/Mods folder

Known Compatabilities:
	Game Version:	Stardew Valley v1.3
	OS:		Windows 10
	SMAPI Version:	2.7

See source code at:
	https://github.com/EKomperud/StardewMods/
