Installion:
1)Install SAMPI for your game.
2)Put the mod folder inside the mods folder(the mods folder is inside your stardew valley folder).

Config file:
The mod config file contain the heart amount value that the mod uses and is 3 by deafult.
If you want you can change it to any number between 0 to 10, and it will change the heart requirment for the dance (0 - 0 hearts needer, 10 - 10 hearts needed.
any number below 0 will count as 0 and any number above 10 will count as 10).
